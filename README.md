# Archipelago

## Organizing standard

File structure

```
playerName = Kebab case of the player name
gameName = Kebab case of the name of the game


Kebab case example
"My long dog" = "myLongDog"
"Risk of Rain 2" = "riskOfRain2"
"Pokemon Red and Blue" = "pokemonRedAndBlue"
```

```
{playerName}
├── {playerName_gameName}.yaml
└── {playerName_gameName}.yaml
```
